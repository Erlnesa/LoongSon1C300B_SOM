## 简介

之前设计了龙芯1C300B的核心板，后来发现大小还能继续压缩，于是有了这个升级版。

![正面](正面.webp)

![反面](反面.webp)

相较于上个版本，依然是四层板。但压缩度更高，尺寸更小。

目前的长宽为：3.82cm X 6.72cm

更换了大多数芯片的封装，把所有的元件全部放置在了PCB正面方便手工焊接。
